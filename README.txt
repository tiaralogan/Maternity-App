# Maternity-App

To run the app Me and Mine:

Requirements:
1. Computer with an Apple environment
2. Latest version of XCode
3. Firebase downloaded

Instructions to simulate the app:
1. Download all the files to the same folder.
2. Navigate to the file "Me and Mine.xcodeproj".
3. Open the file with XCode.
4. Press the play button in the upper left corner of the window
    or  press the "Command" and "R" keys simultaneously.
5. The simulator is now running, and the app will load.
